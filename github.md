
如何使用github

1、安装git
   注：配置环境变量（D:\Program Files\Git\bin）
2、在本地或者在github上新建一个项目
3、本地和github关联：

   （1）从github上新建项目，并clone到本地
       cmd  项目存放路劲
       git remote add origin https://github.com/shenyulin/20150627git.git

   （2）从地址webstrom中push到github
       VCS / 项目右键
       git -->
       commit directory  -->
       git -->
       repository  -->
       push
